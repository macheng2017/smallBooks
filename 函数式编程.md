```js
function add(x, y) {
    return x + y
}
```

```js
function add(x) {
    return function (y) {
            return x + y
    }
}
// 调用方式

add(1)(2)
// 使用这种方式可以减少一个参数

```
```js
function add(x) {
    return function (y) {
            return x + y
    }
}

const inc = add(1)

// 调用方式
inc(2)
// 再来感受下与上面的有什么不同?
```

* 可以通过函数创造函数,通过一个公式,扩展成一个新的公式
* 可以通过这种方式扩展函数的适用范围


```js
R.add(1,3) 

R.add(1)(3)

const inc = R.add(1)
inc(3)
```
* 世界万物是穷举不出来的,函数也是我们是不可能将所有的需求都写成函数
* 但是我们可以退而求次,我们可以通过一个已知的函数可以创造一个新的函数供我们使用
* 柯理化
